================================================================================
IMPORTANT NOTES
================================================================================

1. COLUMN NAMES MUST MATCH EXACTLY
   - Column names are case-sensitive
   - Extra spaces will cause column lookup to fail
   - Use the exact names provided in this document

2. ENUM VALUES ARE CASE-INSENSITIVE
   - "Male", "male", "MALE" all map to: male
   - "don't know", "Don't know", "DON'T KNOW" all map to: dont_know
   - Spaces are auto-normalized

3. ROW FILTERING RULES
   - Leads: Row synced only if "Prospect Name" has a value
   - Clients: Row synced only if "Client Name" has a value
   - Business Opportunities: Row synced only if "Expected Amount" has a value

4. AUTOMATIC DELETION
   - When you sync a sheet, ALL previous rows for that sheet are deleted and replaced
   - This ensures data consistency but means partial syncs will remove old data

5. SYNC BEHAVIOR
   - Manual Sync: Edit a cell → 3 second debounce → webhook syncs only that sheet
   - Full Sync: Menu button syncs all sheets with data
   - Auto-Sync: Install trigger → automatic sync on every cell edit

6. WEBHOOK SECRET
   - All sheets use the same webhook endpoint
   - Secret: your-webhook-secret-bffd0b2d86670879037f23b1e1776793d9bea9f22cbe48f32f675095078cbfb4
   - (This should not be modified by users)

================================================================================
TROUBLESHOOTING
================================================================================

Issue: Data not syncing
Solution: Check that column names match exactly, enum values are from the list above

Issue: Only some rows syncing
Solution: Check if required column (name for Leads/Clients, amount for BO) is empty

Issue: Enum value errors in logs
Solution: Use values from the "Valid Values" list, ensure spelling matches

Issue: Column not recognized
Solution: Verify column header spelling matches exactly (case-sensitive)
